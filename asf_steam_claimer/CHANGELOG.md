# Changelog

## 1.0.0 - 2026-01-09

### Initial Release

- Standalone ASF addon for Steam free games claiming
- Configurable MinReviewScore filtering (1-9 scale)
- Configurable game age filtering (MaxDaysOld)
- Option to ignore free weekend games
- Headless operation with 2FA support
- Web UI on port 1242
- Clean config generation (no FreePackages plugin to avoid config bugs)
