# Changelog

All notable changes to the Spotify Player add-on will be documented in this file.

---

## [1.0.0] - 2025-12-14

### Added

- Initial release of Spotify Player (spotifyd) add-on
- Lightweight Spotify Connect client using spotifyd
- Keeps Spotify session active 24/7 for voice control
- Configurable device name and audio bitrate
- PulseAudio backend for audio subsystem
- Multi-architecture support (amd64, aarch64, armv7)
